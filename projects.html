AI Machine Learning:Mussels Detection using YOLO

I developed an AI-based object detection system to identify mussels in images using the YOLO (You Only Look Once) deep learning model. The main objective of this project was to explore how computer vision techniques can be applied in environmental and aquatic monitoring applications. In this project, I prepared and annotated a custom dataset of mussel images to train the YOLO model. The dataset was carefully labeled to ensure accurate object detection performance. After preprocessing and dataset preparation, I trained the model and evaluated its performance using standard metrics such as precision, recall, and confidence scores. The trained model was tested on unseen images to verify its real-time detection capability. The results showed that the system can successfully detect mussels with good accuracy under different conditions. This project helped me gain practical experience in: Dataset collection and annotation for object detection Training and fine-tuning YOLO models Understanding computer vision workflows Evaluating model performance and improving accuracy The project demonstrates the potential of AI-based vision systems in marine and environmental monitoring applications, and it can be extended for real-time underwater or drone-based detection systems.

Design and Development of High Precision wireless Stethoscope

I designed and developed a wireless digital stethoscope system using Arduino to capture and transmit heartbeat sounds in real time. The system is aimed at improving remote patient monitoring by enabling doctors to listen to heart sounds without being physically present near the patient. The project involves acquiring analog heart sound signals using a sensor module, processing the signal through an Arduino microcontroller, and transmitting the data wirelessly to a receiver for audio playback. The design focuses on improving signal clarity and reducing noise to ensure accurate heartbeat monitoring. This system demonstrates the application of embedded systems in healthcare and telemedicine, especially for remote diagnosis and early health monitoring. Key highlights: Heartbeat signal acquisition and processing using Arduino Wireless transmission of biomedical audio signals Focus on signal clarity and noise reduction Application in remote healthcare and telemedicine systems

Monitoring and Alert System for Cerebral Palsy Walkers Using Image Processing

I developed a smart assistive walker system designed for cerebral palsy patients using ESP32, a camera module, and sensor integration. The system focuses on real-time monitoring of patient movement to enhance safety and provide timely assistance during abnormal conditions. The system uses image processing techniques to detect posture abnormalities and fall events, while additional sensors improve accuracy in motion tracking. When a fall or unsafe posture is detected, the system immediately triggers alerts to caregivers through a connected communication module. It can also activate a safety support mechanism to assist the user during emergencies. This project combines embedded systems, IoT, and computer vision to create a practical healthcare monitoring solution aimed at improving patient independence and safety. Key highlights: Real-time posture and fall detection using ESP32 camera module Sensor-assisted motion monitoring for improved accuracy Instant alert system for caregivers via IoT communication Safety mechanism activation during emergency conditions Application in assistive healthcare and rehabilitation systems
